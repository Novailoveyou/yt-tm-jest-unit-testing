## Commands run in order

Init package.json

```zsh
pnpm init
```

Install dev dependencies

```zsh
pnpm add typescript ts-node @types/node -D
```

Init tsconfig.json

```zsh
pnpm exec tsc --init
```

Install Jest

```zsh
pnpm add jest -D
```

Run tests

```zsh
pnpm test
```
